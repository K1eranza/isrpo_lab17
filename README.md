## Лабораторная работа №17. Основы алгоритмов и блок-схем
### Цель лабораторной работы: 
* Понять, что такое алгоритм и его свойства
* Научиться составлять блок-схемы алгоритмов
* Освоить работу с расширением Draw.io в VS Code
* Научиться переводить блок-схемы в код на языке C#
* Решать практические задачи с использованием алгоритмов
### 1. Работа с расширением Draw.io в VS Code
**Draw.io Integration** создание блок-схем прямо в Visual Studio Code. Это позволяет визуализировать алгоритмы перед написанием кода.

**Основные элементы блок-схем:**
- **Овал** — начало или конец 
- **Прямоугольник** — действие 
- **Ромб** — условие 
- **Параллелограмм** — ввод/вывод данных
- **Стрелки** — направление выполнения

### 2. Составление блок-схем алгоритмов

- Приготовление чая (линейный алгоритм)
- Выбор одежды по погоде (if-else)
- Калькулятор чаевых (переменные и вычисления)
- Определение оценки (множественное ветвление if-else if-else)
- Подъём по лестнице (цикл while)
- Таблица умножения (цикл for)

### 3. Перевод блок-схем в код на C#

**Базовые конструкции:**
- Ввод/вывод данных (`Console.WriteLine`, `Console.ReadLine`)
- Переменные и типы данных (`int`, `double`, `string`, `bool`)
- Условные операторы (`if`, `else if`, `else`)
- Циклы (`while`, `for`)
- Массивы для хранения коллекций данных
- Функции (методы) для повторного использования кода
